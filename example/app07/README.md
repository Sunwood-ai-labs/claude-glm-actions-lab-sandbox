# おみくじアプリ 🔮

シンプルで楽しいおみくじWebアプリケーション。グラデーション背景とアニメーションで、運勢を占う体験をより魅力的にします。

## 概要 📖

このアプリケーションは、日本伝統のおみくじを現代的なWeb技術で再現したものです。ボタンをクリックするだけで、6種類の運勢からランダムに結果を表示し、過去の履歴も確認できます。

## 機能 ✨

- **ランダムおみくじ**: ボタン一つで運勢を占えます
- **6種類の運勢**: 大吉、中吉、小吉、吉、末吉、凶の結果が表示されます
- **シェイクアニメーション**: おみくじを引く際のシェイク演出で期待感を高めます
- **履歴機能**: 過去10件の結果を時刻とともに記録・表示します
- **モダンなUI**: グラデーション背景とカードデザインで見やすいインターフェース

## 使用方法 🚀

### インストール

特別なインストールは不要です。以下の手順で使用できます。

```bash
# リポジトリをクローン
git clone https://github.com/Sunwood-AI-OSS-Hub/claude-glm-actions-lab-sandbox.git

# ディレクトリに移動
cd claude-glm-actions-lab-sandbox/example/app07
```

### 実行

1. `index.html` をブラウザで開きます
2. 「おみくじを引く」ボタンをクリックします
3. シェイクアニメーション後に結果が表示されます
4. 過去の結果は「過去の結果」セクションで確認できます

## 運勢の確率 📊

重み付けランダム選択により、以下の確率で運勢が決まります：

| 運勢 | 確率 |
|------|------|
| 大吉 | 5% |
| 中吉 | 20% |
| 小吉 | 25% |
| 吉 | 25% |
| 末吉 | 15% |
| 凶 | 10% |

# サイコロアプリ 🎲

シンプルで美しい3Dサイコロアプリ。ボタンをクリックするだけで、リアルなサイコロの転がるアニメーションと共にランダムな結果（1〜6）が表示されます。

## 機能

- **3Dアニメーション**: サイコロがリアルに回転するアニメーション
- **ランダム結果**: 1〜6の目をランダムに生成
- **履歴機能**: 過去10回分の結果を時刻と共に記録
- **レスポンシブデザイン**: PCでもスマートフォンでも綺麗に表示

## 使い方

1. `index.html` をブラウザで開く
2. 「サイコロを振る」ボタンをクリック
3. サイコロが回転し、結果が表示されます
4. 履歴は下に自動的に記録されます

## 技術的なポイント

- CSS 3D Transformsを使用したサイコロの表現
- JavaScriptによる動的な回転角度の計算
- アニメーション完了後のコールバック処理
- 履歴の配列管理とDOM操作

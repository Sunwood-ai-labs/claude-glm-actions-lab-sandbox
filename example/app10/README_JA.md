# おみくじ 🎴

シンプルで美しいおみくじアプリケーション。今日の運勢を占いましょう！

## 概要 📖

このアプリケーションは、日本伝統のおみくじをデジタルで体験できるツールです。大吉から大凶までの6種類の運勢の中から、シャッフルアニメーションと共にランダムに結果が表示されます。

## 機能 ✨

- **6種類の運勢**: 大吉、中吉、小吉、末吉、凶、大凶
- **シャッフルアニメーション**: おみくじを引いている臨場感のあるアニメーション
- **運勢メッセージ**: 各運勢に対応した励ましのメッセージ
- **美しいデザイン**: グラデーション背景と和風カラースキーム
- **レスポンシブ対応**: モバイルデバイスでも快適に動作

## 使い方 🚀

### インストール

特別なインストールは不要です。以下の手順で実行できます：

```bash
# リポジトリをクローン
git clone https://github.com/Sunwood-AI-OSS-Hub/claude-glm-actions-lab-sandbox.git

# ディレクトリに移動
cd claude-glm-actions-lab-sandbox/example/app10
```

### 実行方法

1. `index.html` をWebブラウザで開く
2. 「おみくじを引く」ボタンをクリック
3. シャッフルアニメーション後、運勢結果が表示されます
4. 「もう一度引く」ボタンで何度でも占えます

## 運勢の種類 🍀

| 運勢 | 説明 |
|------|------|
| 大吉 | 最高の一日になるでしょう！何をやってもうまくいきそうです！ |
| 中吉 | 良い日になるでしょう！前向きに取り組めば成功します！ |
| 小吉 | 小さな幸せが訪れるでしょう。日常を楽しみましょう！ |
| 末吉 | 努力が実を結ぶのはもう少し後。諦めずに続けましょう！ |
| 凶 | 注意が必要な日です。慎重に行動しましょう。 |
| 大凶 | 今日は大人しく過ごすのが吉。明日はきっと良い日です！ |
